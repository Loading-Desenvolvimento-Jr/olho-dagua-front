## ğŸ¨ Design System

We use a custom "Neo-Brutalism" inspired design system with specific tokens configured in `globals.css`.

* **Fonts:**
    * Titles: `Londrina Solid` (var: `--font-londrina`)
    * Body: `Livvic` (var: `--font-livvic`)
* **Key Colors:**
